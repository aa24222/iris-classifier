# Iris Classifier ML Project

This project demonstrates a machine learning workflow using the Iris Dataset. It includes:

- Loading and visualizing data using pandas and Matplotlib.
- Feature scaling
- Training a Random Forest Classifier
- Model evaluation (accuracy, classification report)
- Making predictions on new samples

## Dataset

The project uses the Iris dataset from scikit-learnâ€™s built-in datasets:

- 150 samples of iris flowers
- 4 features: sepal length, sepal width, petal length, petal width
- 3 classes: setosa, versicolor, virginica
- No external CSV file is required
